# Create PETT App

- Preact
- ESBuild
- TailwindCSS
- Typescript

Create PETT App scaffolds a simple directory for an app using the above technologies, so you don't have to deal with setting up the environment yourself

## How to Use

```bash
npx @ekwoka/create-pett-app [dirname] [options]
```

Simple!!!

## Options

Package Manager:
  `--package-manager`
  - Default: `npm`
  - Acceptable Options:
    - `npm`
    - `pnpm`
    - `yarn`
